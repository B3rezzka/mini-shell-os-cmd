# Python mini-shell os cmd, made for lab #2 MAI

## Введение
**MiniShell** — это простая командная оболочка, написанная на Python, которая поддерживает ограниченный набор команд, аналогичных тем, что используются в Unix-подобных системах. Она может использоваться для выполнения базовых операций с файлами и каталогами.

## Возможности

- Поддержка команд:
  - `cd` — смена каталога
  - `ls` — список файлов и каталогов
  - `ls -l` — подробный список (как в настоящем `ls -l`)
  - `cat` — вывод содержимого файла
  - `cp` — копирование файлов и каталогов (с опцией `-r`)
  - `mv` — перемещение или переименование файлов и каталогов
  - `rm` — удаление файлов и каталогов (с опцией `-r`)
  - `exit` — выход из оболочки

- Поддержка **абсолютных и относительных путей**.
- **Логирование всех команд** в файл `shell.log`.

- Допущения и ограничения

 - Команды должны вводиться через пробел
 - Поддерживаются абсолютные и относительные пути
 - Работа на Linux и Mac OS
 - Для путей с пробелами используйте кавычки: cd "папка с пробелами" (для путей, в которых используются файлы/каталоги с пробелами)
 - Все операции логируются в файл src/shell.log

## Ошибки
Все ошибки обрабатываются и также логируются, выводятся в случае возникновения.
